## Panduan Penggunaan Aplikasi

1. **Clone Repository**
    ```bash
    git clone git@github.com:dedimujahid/sekawan-media.git
    ```

2. **Buka Folder Aplikasi di Text Editor**

3. **Install Dependencies**
    ```bash
    npm install
    ```

4. **Jalankan Proyek**
    ```bash
    npm start
    ```
